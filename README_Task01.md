# Task‑01: Distribution Visualisation (Population, Total)

**Dataset:** World Bank “Population, total” indicator  
**File:** API_SP.POP.TOTL_DS2_en_csv_v2_38144.csv

## Visualisations included:
- Histogram of population across countries in 2020 (`hist_population_2020.png`)
- Bar chart of top 10 most populated countries in 2020 (`bar_top10_population_2020.png`)

## How to run
1. Open `Task01_visualization.ipynb` in Google Colab or locally.
2. Run all cells.
3. The plots will be displayed and saved as PNG files in the working directory.
